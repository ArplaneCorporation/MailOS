MailOS

Idea date: 2023, Saturday, December 2nd

An embedded operating system designed for email and also physical mail/packaging management

Can also be used to host a mail server

Offline mode: browse your emails offline

Default client: DMAIL

Written in: C, Rust, D, PHP
